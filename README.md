# mejiro voice clip

音声・動画ファイルからクリップを切り出してエクスポートするデスクトップアプリケーションです。

## スタック

| レイヤー | 技術 |
|---|---|
| フレームワーク | [Tauri v2](https://tauri.app/) + [SvelteKit](https://kit.svelte.dev/) |
| 波形表示 | [WaveSurfer.js](https://wavesurfer.js.org/) |
| 音声・動画処理 | [ffmpeg](https://ffmpeg.org/) (ffmpeg-sidecar 経由) |

## 主な機能

### ファイル読み込み・波形表示

- ファイルダイアログまたはウィンドウへの **ドラッグ＆ドロップ** でファイルを開く
- 対応形式: `mp4` / `mkv` / `mov` / `avi` / `webm` / `flv` / `ts` / `m4v` / `mp3` / `wav` / `aac` / `flac` / `ogg` / `m4a` / `opus` / `wma`
- 動画ストリームは Tauri カスタムプロトコル (`http://stream.localhost/`) 経由でシークをサポート

### ffmpeg セットアップ

アプリ起動時にシステム PATH → サイドカーバイナリの順で ffmpeg を自動検出します。
見つからない場合はセットアップダイアログが開きます。

- **パスを指定**: インストール済みの ffmpeg.exe を手動で選択
- **自動ダウンロード**: ffmpeg-sidecar によりアプリフォルダへ自動取得（プログレスバー表示付き）

### リージョン編集

#### Add Mode
 1. 開始点を設定
 2. 終了点を設定
 3. Tagを選択（任意）
 4. Region名を入力
 5. 追加

#### Edit Mode
 1. リージョンを選択
 2. 任意のフィールドを編集
 3. 保存

#### Tag機能
- タグを作成してリージョンに紐付け
- タグでリージョンをフィルタリング

### エクスポート

- リージョンをまとめて一括エクスポート
- 出力フォーマットを選択:
  - **WAV** — 音声のみ (`pcm_s16le` / 44100 Hz)
  - **MP4** — 動画クリップ (`libx264` + `aac` / faststart)

### キーボードショートカット

任意のキーをショートカットに指定できます
